# Annet

Anonymous real-time chat. Zero login.

## Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Socket.IO client

## Dev

```bash
npm install
npm run dev
```

Runs on `http://localhost:8080`. Requires the backend running on `http://localhost:3001`.

## Backend

See `/server` for the Node.js + Express + Socket.IO backend.
